# Prettify JSON

Program prints pretty json from transmitted file.

# Quickstart

Script can be run from command line.
Example of script launch on Linux, Python 3.5:

```bash
$ python pprint_json.py <path to file>
```
Output example:
```bash
[
    {
        "coordinates": [
            37.58803599964753,
            55.89020100016689
        ],
        "type": "Point"
    }
]
```
